# Roadmap da Plataforma

## Fase 1. Fundacao do modelo

- Normalizar a estrutura dos questionarios
- Definir escalas, secoes, itens e dimensoes
- Mapear regras de pontuacao e reversao

## Fase 2. Biblioteca completa

- Cadastrar todos os instrumentos do levantamento
- Separar por versao, publico e contexto de uso
- Identificar questionarios com subescalas

## Fase 3. Motor de respostas e calculos

- Validacao de campos obrigatorios
- Pontuacao automatica
- Interpretacao por faixa e por dimensao
- Exportacao de resultados individuais

## Fase 4. Backend e persistencia

- Autenticacao
- Banco de dados
- API para formularios, respostas e relatórios

## Fase 5. Operacao e seguranca

- Perfis de acesso
- Consentimento
- Auditoria
- Retencao e exportacao de dados
